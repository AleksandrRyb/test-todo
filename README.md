# Инструкция по запуску проекта

- Запустите сервер с помощью команды ```npm run server``` или ```yarn server```
- Запустите клиент с помощью команды ```npm start``` или ```yarn start```

> Сейчас сервер запускается на порте 5555, если этот порт занят то поменяйте порт в package.json в команде запуска сервера, после смены порта не забудте поменять переменную ```PORT``` в по пути ```constants/api-constants```
